# Fun Java CLI
[개발자 속담](https://fun.okdevtv.com/)을 터미널에서 보자.

https://github.com/kenu/funjava

## REQUIRES
- Python > 3.0

## RUN
```sh
py src/fjcli
```

## INSTALL
```sh
python -m pip install --upgrade fjcli
py -m pip install --upgrade fjcli
```

## Usage
```sh
python -m fjcli
py -m fjcli
```
